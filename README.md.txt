# Real Estate Price Prediction using Machine Learning

## Overview

This project predicts housing prices using Machine Learning models such as Linear Regression and Random Forest.

## Features

* Data preprocessing and cleaning
* Missing value handling
* One-hot encoding of categorical variables
* Model training and evaluation
* Model comparison using RMSE
* Feature importance visualization

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Jupyter Notebook

## Results

* Linear Regression RMSE: 52,266
* Random Forest RMSE: 28,873
* Random Forest performed significantly better

## Key Learning

* Data preprocessing techniques
* Regression modeling
* Feature engineering
* Model evaluation
